# moa-gen

> Microservice to get approval certificates

```
https://moa-gen.now.sh/?name=Excellence&receiver=fmal
```

It is possible to change size by adding the `size` parameter:

```
https://moa-gen.now.sh/?receiver=fmal&size=500
```
